# PWS Asset Site

A lightweight, organized React + Vite shell around the bundled single-file PWS asset app export.

## Repository structure

- `src/` — React application code.
  - `components/` — reusable UI components.
- `OkaVlhpisa-V5.2.1.html` — bundled standalone app export.
- `vite.config.js` — Vite build configuration.

## Getting started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
